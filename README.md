# Enterprise Workforce Management Platform

A Spring Boot based workforce management system for managing employees, departments, attendance, and leave requests.

## Tech Stack
- Java 17
- Spring Boot 3.5
- Spring Data JPA
- Spring Security
- MySQL
- Maven
- Swagger OpenAPI

## Features
- Employee Management
- Department Management
- Attendance Tracking
- Leave Management
- Validation
- Global Exception Handling
- REST APIs
- Swagger Documentation

## Project Structure

```text
Department
Employee
Attendance
Leave Request
```
